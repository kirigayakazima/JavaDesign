# spring boot红包

# Controller注解

注解 作用
@Controller 处理http请求
@RestController Spring4之后新加的注解，原来返回JSON需要@ResponseBody配合@Controller
@RequestMapping 配置url映射
@GetMapping get请求路径
@PostMapping post请求路径
@PathVariable 获取url中的数据
@RequestParam 获取请求参数的值

# RESTful API设计

请求类型 请求路径 功能
GET /luckymoenys 获取红包列表
POST /luckymoneys 创建一个红包
GET /luckymoneys/id 通过id查询红包
PUT /luckymoneys/id 通过id更新红包
第一次连接数据库出错显示：Caused by: org.hibernate.HibernateException: Access to DialectResolutionInfo cannot be null when 'hibernate.dialect' not set

解决方案：修改mysql的方言

database-platform: org.hibernate.dialect.PostgreSQLDialect

# yml配置


 















spring:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    #添加serverTimezone=UTC设置时区，以顺便加一个characterEncoding=utf-8
    url: jdbc:mysql://localhost:3306/luckymoney?serverTimezone=UTC
    username: root
    password: 123456

  jpa:
    hibernate:
      #配置库是create还是update
      ddl-auto: create
    #在控制台执行的显示sql语句
    show-sql: true
    #调整数据库的方言，对应不同数据库的
    database-platform: org.hibernate.dialect.MySQL5Dialect
